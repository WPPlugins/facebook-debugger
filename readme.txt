Debugging Sharing to Facebook.

== Description ==

Just click on FB Debugger in the WP Admin Bar and clear Facebook cache for posts or pages.
Very simple configuration.

Visit plugin site: http://www.darkomitrovic.com/wp-plugin/facebook-debug/
 
== Installation ==

1. Unzip the file
2. Upload it to `/wp-content/plugins/` directory
3. Activate it from the plugins section
